Pinning of Columns

Description:

This module provides column pinning to the end user via menu options in the column header

One may want to see details of a row based on certain columns. For example, keep the quoted rate column for a Line item as pinned column and then scroll through all bidders to see Delivery date of the item.

Demo:

Code:

<div ui-grid="gridOptions" class="grid" ui-grid-pinning></div>
